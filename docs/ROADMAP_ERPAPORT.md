# Roadmap E-Raport

## Prioritas Berikutnya

1. Governance period: enforce rule `planned/active/closed` + 1 current.
2. Academic engine: nilai + absensi + validasi berbasis periode aktif.
3. Report card engine: kalkulasi, lock final, publish ke murid.
4. Teacher workflows: input nilai/absen per kelas-mapel yang di-assign.
5. Student/parent view: baca raport/riwayat per periode.
6. Hardening: audit log, e2e tests, backup/export.
